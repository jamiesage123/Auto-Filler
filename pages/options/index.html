Auto Filler

Auto Filler allows you to automatically fill in a form with dummy or preset data with full customisation.

Below, you can find all of the settings for Auto Filler.

You can add a new or edit an existing Auto Filler rule here.

Usage

Use the following key combinations to activate Auto Filler.

Please note that Auto Filler only activates when you are focused on a field.

Key Combination Action
CTRLSHIFTF Fill in the current form using your settings
CTRLALTF Fill in the current element using your settings

To customise your key combinations, head over to chrome://extensions/ and find "Keyboard shortcuts" at the bottom of the page
